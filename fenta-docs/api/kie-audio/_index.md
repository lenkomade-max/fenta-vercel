# KIE.ai Audio APIs

## Base URL
```
https://api.kie.ai
```

## Authentication
```
Authorization: Bearer YOUR_API_KEY
```

## Available Models

| Model | Endpoint | Credits |
|-------|----------|---------|
| ElevenLabs TTS | `/api/v1/elevenlabs/tts` | 5-10/min |
| ElevenLabs SFX | `/api/v1/elevenlabs/sfx` | 5-10 |
