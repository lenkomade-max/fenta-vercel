# ЗАДАЧА ДЛЯ АГЕНТА: Создать 10 промптов для сбора баз данных

## КОНТЕКСТ ПРОЕКТА

**Fenta** — SaaS платформа для автоматического создания коротких видео (TikTok, Reels, Shorts).

Ядро продукта — **мульти-агентная AI система** из 20 агентов, которые создают вирусный контент.

Чтобы агенты работали качественно, им нужны **10 баз данных (RAG)** с примерами, правилами и шаблонами.

---

## ТВОЯ ЗАДАЧА

Создать **10 файлов с промптами** в папке `/root/fenta-vercel/prompts-for-bd/`:


## ДЕТАЛИ ПО КАЖДОЙ БАЗЕ

### 1. hooks (хуки)
- **Цель:** Вирусные первые 3 секунды видео
- **Агент:** Script Agent, Hook Agent
- **Нужно:** 500+ хуков, разбитых по жанрам (true-crime, facts, how-to, news, etc.)
- **Формат:** текст хука + жанр + тип (curiosity, shock, question, pov) + score (0-100)
- **Источники:** TikTok viral videos, YouTube Shorts, копирайтинг блоги

### 2. scripts (сценарии)
- **Цель:** Примеры готовых сценариев для обучения
- **Агент:** Script Agent, Rewrite Agent
- **Нужно:** 300+ сценариев 30-60 секунд
- **Формат:** полный текст + жанр + структура + длина + платформа
- **Источники:** Транскрипты вирусных видео, сценарные курсы

### 3. structures (структуры)
- **Цель:** Формулы построения сценария для каждого жанра
- **Агент:** Orchestrator, Script Agent
- **Нужно:** 30 жанров × детальная структура
- **Формат:** жанр + beats (Hook→Context→Twist→Reveal→CTA) + timing + examples
- **Источники:** Storytelling frameworks, viral content analysis

### 4. video_prompts (промпты для видео)
- **Цель:** Работающие промпты для Sora, Kling, Runway, Minimax
- **Агент:** Video Prompt Agent
- **Нужно:** 200+ промптов с результатами
- **Формат:** промпт + модель + настройки + жанр + стиль
- **Источники:** OpenAI Cookbook, Reddit r/sora, Discord серверы

### 5. cinematography (кинематография)
- **Цель:** Правила камеры, света, движения для каждого жанра
- **Агент:** Video Prompt Agent
- **Нужно:** Детальные правила для 30 жанров
- **Формат:** жанр + camera (angles, movements) + lighting + color grading + motion
- **Источники:** Film school resources, cinematography guides, YouTube tutorials

### 6. styles (визуальные стили)
- **Цель:** Эстетики и визуальные направления
- **Агент:** Style Agent, Video Prompt Agent, Image Prompt Agent
- **Нужно:** 50+ стилей с примерами
- **Формат:** название + описание + color palette + texture + mood + example prompts
- **Источники:** Midjourney styles, design blogs, aesthetic guides

### 7. music_moods (музыка)
- **Цель:** Соответствие жанра контента → музыкальным параметрам
- **Агент:** Music Agent
- **Нужно:** 30 жанров × музыкальные правила + 100 Suno промптов
- **Формат:** жанр + BPM + key + mood + energy + Suno prompt examples
- **Источники:** Music theory, Suno documentation, viral video analysis

### 8. voice_profiles (голоса)
- **Цель:** Голоса ElevenLabs и их применение
- **Агент:** Voice Agent
- **Нужно:** Все голоса ElevenLabs + когда какой использовать
- **Формат:** voice_id + name + characteristics + best_for_genres + settings
- **Источники:** ElevenLabs documentation, voice acting guides

### 9. platform_rules (правила платформ)
- **Цель:** Специфика TikTok, Reels, Shorts, YouTube
- **Агент:** Все агенты
- **Нужно:** Детальные правила каждой платформы
- **Формат:** платформа + aspect_ratio + duration + hooks + captions + hashtags + algorithm
- **Источники:** Официальные гайды, SMM блоги, creator programs

### 10. viral_rules (правила вирусности)
- **Цель:** Научные и практические правила создания вирусного контента
- **Агент:** Quality Agent, Script Agent
- **Нужно:** Исследования + практические правила
- **Формат:** rule + explanation + examples + source
- **Источники:** Academic research, Jonah Berger "Contagious", viral marketing studies

## КРИТИЧЕСКИ ВАЖНО

1. **Промпты должны быть МАКСИМАЛЬНО детальными** — агент который получит промпт должен понимать ТОЧНО что искать

2. **Формат JSON обязателен** — база данных будет в векторном хранилище, нужна структура

3. **Источники конкретные** — не "искать в интернете", а конкретные сайты, каналы, документации

4. **Качество > Количество** — лучше 100 отличных примеров чем 1000 плохих

5. **Метаданные важны** — жанр, платформа, score, source — всё нужно для фильтрации
